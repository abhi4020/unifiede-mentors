# **App Name**: BlogFlow

## Core Features:

- Blog Layout: Clean and organized display of blog posts.
- Blog Posts: Display of individual blog posts with titles, content, and metadata.
- Navigation: Easy navigation between different sections of the blog.
- Comment Section: Allow users to post comments on blog entries. Only allow comments from authenticated users (no database component - leverage browser cookies)
- Responsiveness: Ensure the blog is responsive and accessible on different devices and screen sizes.
- Search Bar: Enable users to search for specific blog posts by keywords. Use an LLM tool to understand and return similar words.
- Social Media Sharing: Add buttons for users to easily share blog posts on social media platforms.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a calm and inviting atmosphere.
- Background color: Light gray (#F0F0F0) to ensure readability and reduce eye strain.
- Accent color: Dusty rose (#D8BFD8) for interactive elements such as links and buttons.
- Body and headline font: 'Literata', a transitional serif with a literary and slightly formal feel.
- Simple, minimalist icons for categories and social media sharing buttons.
- Clean and well-organized layout with a clear hierarchy to improve readability.
- Subtle transition effects and animations on interactive elements for a smooth user experience.
